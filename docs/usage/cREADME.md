# Usage

## Example Flow
1. Create a `CodeGenerator` using a `CodeTemplateFactory`.
2. Register templates with the `TemplateManager` singleton.
3. Construct prompts with `PromptEngine` and a `PromptStrategy`.
4. Validate generated code using `CodeValidator` with a `ValidationStrategy`.

Refer to [cREADME.md](../../cREADME.md) for a complete example.
